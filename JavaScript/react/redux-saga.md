## 리덕스 사가란?
- 잔망스러운 비동기 동작(데이터 fetching, 브라우저 캐시 접근)을 쉽게 만들기
- 사가: 애플리케이션에서 사이드 이펙트만을 담당하는 별도 쓰레드.
- redux saga: 리덕스 미들웨어라서 쓰레드가 메인 A에서 일반적인 리덕스 액션을 통해 실행되고, 멈추고 취소. 그리고 모든 리덕스 A의 상태에 접근 가능하고 액션 디스패치
